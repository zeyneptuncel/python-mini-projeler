# 🏦 Sistem Simülasyonları

Gerçek hayat senaryolarını kodlara döktüğüm, `while` döngüleri ve menü akışlarıyla kurgulanmış etkileşimli konsol simülasyonları.

**Neler Var?**
* `atm_simulation.py`: Kullanıcı bakiye sorgulama, para yatırma ve çekme işlemlerini barındıran ATM simülasyonu.
* `restaurant_app.py`: Müşteri sipariş alma, sepet yönetimi (ekleme/silme) ve hesap tutarı hesaplama süreçlerini içeren restoran otomasyonu.
