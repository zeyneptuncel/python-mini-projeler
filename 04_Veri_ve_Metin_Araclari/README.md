# 📝 Veri ve Metin İşleme Araçları (String Methods)

Python'daki string (metin) metotlarını ve mantıksal doğrulama (validation) işlemlerini test ettiğim ufak araçlar.

**Araçlar:**
* `password_control.py`: Metinde büyük/küçük harf sayı olup olmamasına göre şifre oluşturucu.
* `metin_araci.py`: Metinler üzerinde arama, değiştirme (replace) ve formatlama işlemleri yapan string  aracı.
