# 🗂️ Dosya Yönetimi (File Handling) Projeleri

Bu klasörde verileri kalıcı olarak hafızada tutmak için `.txt` dosyaları kullanarak yazdığım "Backend" temelli sistemler bulunuyor.

**Projeler:**
* `kullanici_sistemi.py`: Dosya tabanlı çalışan Kullanıcı Kayıt ve Giriş (Auth) mimarisi.
* `not_uygulamasi.py`: Öğrenci notlarını kaydeden, okuyan ve harf notu hesaplayan sistem.
